### Tecnologías utilizadas
- Python 3.x  
- OpenCV (cv2)  
- matplotlib  
- NumPy  
- watermark (para información del entorno)  

---

### Se trata de:
**Contenido del Notebook**
- Exploración del concepto de **color** en imágenes digitales.
- Lectura de imagen a color desde una URL utilizando OpenCV.
- Conversión de formatos de color: de BGR a RGB y a escala de grises.
- Separación de canales de color (Rojo, Verde, Azul).
- Visualización individual de cada canal y comparación con la imagen original.
- Conversión a espacio de color HSV y visualización de sus componentes (Tono, Saturación, Valor).
- Reflexión sobre cómo los distintos espacios de color representan la información visual.
- Discusión sobre posibles aplicaciones del análisis por canales y espacios de color.

---

### Instrucciones de uso
1. Asegurate de tener Python 3 instalado.  
2. Instalá las dependencias necesarias ejecutando:  
   ```
   pip install opencv-python matplotlib watermark
   ```  
3. Abrí el notebook con Jupyter Notebook, Deepnote o Google Colab.  
4. Ejecutá las celdas una a una para observar cómo se manipula la información de color.  
5. Podés reemplazar la URL de la imagen para trabajar con otras imágenes de prueba.

---

### Este repositorio se usa para:
**Trabajo de Laboratorio: Edición y Procesamiento de Imágenes**  
El objetivo del notebook es comprender cómo se codifica el color en imágenes digitales, explorando los canales RGB y HSV, y visualizando cómo cada uno de ellos aporta información distinta al análisis visual.

---

### Actividades sugeridas
- Probar con imágenes que tengan predominancia de un solo color.  
- Comparar visualmente los canales RGB y HSV.  
- Aplicar umbrales a cada canal para segmentar regiones específicas.  
- Investigar en qué contextos es más útil trabajar en HSV que en RGB.  
- Realizar modificaciones en un solo canal y observar los efectos en la imagen combinada.
